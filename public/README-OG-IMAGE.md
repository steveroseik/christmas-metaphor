# Open Graph Image Setup

For WhatsApp and other social media platforms to display a proper thumbnail, you need a PNG image at `/public/og-image.png` with dimensions 1200x630 pixels.

## Quick Setup Options:

### Option 1: Use the HTML Generator
1. Open `public/generate-og-image.html` in your browser
2. Take a screenshot or use a tool like:
   - [htmlcsstoimage.com](https://htmlcsstoimage.com)
   - [screenshot.rocks](https://screenshot.rocks)
   - Browser DevTools (take screenshot of the page)
3. Save as `og-image.png` in the `public` folder
